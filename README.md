# Приложение "Заметки" с PWA функционалом

## Описание проекта

Данное приложение представляет собой менеджер заметок с полным PWA функционалом, включая:
- Офлайн-доступ через Service Worker
- Установку на устройство через Web App Manifest
- HTTPS соединение для безопасной работы
- Архитектуру App Shell для мгновенной загрузки
- WebSocket для двусторонней связи в реальном времени
- Push-уведомления с возможностью откладывания

## Выполненные практические задания

### Практическое занятие №13 - Service Worker
- Реализована регистрация Service Worker
- Кэширование статических ресурсов
- Работа приложения в офлайн-режиме
- Сохранение заметок в localStorage

### Практическое занятие №14 - Web App Manifest
- Создан файл manifest.json с полными метаданными
- Добавлены иконки разных размеров
- Настроены мета-теги для мобильных платформ
- Возможность установки приложения на устройство

### Практическое занятие №15 - HTTPS + App Shell
- Настроен HTTPS через mkcert
- Реализована архитектура App Shell
- Динамическая загрузка контента через fetch
- Стратегия кэширования Network First для динамических страниц
- Добавлена страница "О приложении"

### Практическое занятие №16 - WebSocket + Push
- Установлены зависимости: express, socket.io, web-push, body-parser, cors
- Сгенерированы VAPID-ключи
- Реализован сервер на Node.js
- Интегрирован Socket.IO для двусторонней связи
- Добавлена подписка на push-уведомления
- Кнопки включения/отключения уведомлений

### Практическое занятие №17 - Детализация Push
- Добавлена форма для создания заметок с напоминанием
- Поле выбора даты и времени (datetime-local)
- Планирование push-уведомлений на сервере с setTimeout
- Кнопка "Отложить на 5 минут" в уведомлениях
- Обработка действия snooze через Service Worker
- Хранение активных таймеров в Map на сервере
